# Marketing AI Quantization Suite

이 저장소는 **Gemma 4 (e2b)** 기반의 마케팅 특화 AI 모델을 **GGUF (Q4_K_M)** 형식으로 양자화하여, 일반 PC 환경에서도 효율적으로 실행할 수 있도록 최적화하는 스크립트 모음입니다.

## 📌 주요 기능
- **고효율 양자화:** Hugging Face에서 학습된 BF16 모델을 불러와 Q4_K_M(4-bit)으로 압축합니다.
- **멀티모델 지원:** 텍스트와 이미지를 동시에 처리하는 멀티모달 모델 최적화 기능을 포함합니다.
- **로컬 실행 최적화:** 최신 Gemma 4 기반 모델을 개인 워크스테이션에서 빠르게 구동할 수 있습니다.

## 🛠 사용 방법
본 스크립트는 다음 환경에서 제작 및 테스트되었습니다.
1. **Google Colab:** Unsloth 기반의 학습/병합 환경
2. **llama.cpp:** GGUF 변환 및 양자화 엔진
